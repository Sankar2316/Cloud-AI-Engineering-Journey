## ☁️ What is Cloud Computing?

Cloud computing is the delivery of computing services such as servers, storage, databases, networking, and software over the internet.

### 🔑 Key Features

* On-demand access
* Scalability
* Pay-as-you-go pricing
* High availability

---

## 🌍 Major Cloud Providers

* Amazon Web Services (AWS)
* Microsoft Azure
* Google Cloud Platform (GCP)

---

## 🧱 Types of Cloud Services

### 1. Infrastructure as a Service (IaaS)

Provides virtual machines, storage, and networking.

### 2. Platform as a Service (PaaS)

Provides a platform to build and deploy applications.

### 3. Software as a Service (SaaS)

Provides ready-to-use applications over the internet.

---

## 🤖 What is Artificial Intelligence (AI)?

Artificial Intelligence is the ability of machines to simulate human intelligence such as learning, reasoning, and decision-making.

---

## 🧠 Types of AI

* **Narrow AI** → Task-specific (e.g., chatbots)
* **General AI** → Human-level intelligence (future concept)

---

## 🔗 Cloud + AI Integration

Cloud platforms enable AI by providing:

* Scalable storage
* High-performance computing
* Pre-built AI services
* Easy deployment
